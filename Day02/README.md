## Problem Solved
**FizzBuzz (LeetCode)**

## Description
The FizzBuzz problem requires generating output for numbers from 1 to *n* based on divisibility rules:
- Print **"FizzBuzz"** if a number is divisible by both 3 and 5
- Print **"Fizz"** if divisible by 3
- Print **"Buzz"** if divisible by 5
- Otherwise, print the number itself

## Approach
- Iterated from 1 to *n* using a loop
- Applied conditional checks in the correct order
- Stored results in a list using Java’s `ArrayList`
- Returned the final list as output
